# Price Calculator 💰

A simple JavaScript project that calculates the final price of a product after applying tax and discount.

## 📌 Features
- Calculate product price
- Add tax
- Apply discount
- Return the final price

## 🧠 Concepts Used
This project practices basic JavaScript concepts:

- Functions
- Parameters
- Return values
- Variables
- Basic arithmetic operations

## 💡 Example Output

Phone Final Price: 1050  
Laptop Final Price: 2050  
Headphones Final Price: 310
Elctric Scooter Final Price: 480

Playstation4 Final Price: 720

Playstation5 Final Price: 1600
